Time profiles

To provide data for different time structures there is a flexible system of different time profiles that can be indexed by time periods.

  • FixedProfile: Time profile with the same value for all time periods
  • OperationalProfile: Time profile with values varying with operational time periods
  • ScenarioProfile: Holds a separate time profile for each operational scenario
  • RepresentativeProfile: Holds a separate time profile for each representative period
  • StrategicProfile : Holds a separate time profile for each strategic period
  • StrategicStochasticProfile : Holds a separate time profile for each strategic node in a strategic tree

The following code example shows how these profile types can be combined in a flexible manner to produce different overall profiles.

rep_periods = RepresentativePeriods(2, 365, [0.6, 0.4], [SimpleTimes(7,1), SimpleTimes(7,1)])
periods = TwoLevel(2, 365, rep_periods)

cost = StrategicProfile(
            [
                RepresentativeProfile(
                    [
                        OperationalProfile([3, 3, 4, 3, 4, 6, 5]),
                        FixedProfile(5)
                    ]
                ),
                FixedProfile(7)
            ]
        )
